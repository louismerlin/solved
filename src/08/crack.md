# GLASSHEADS, pt. 1 Crack the \*\*\*\*\*\*

This challenge can be found on the inside of the cover of issue 8.

We see a list of 22 words, and a URL that points to a hint:

> Hint: the word list in this code is sometimed called "biometric".

Looking-up "biometric word list" on my search engine points me to [this page about "PGP word list"](https://en.wikipedia.org/wiki/PGP_word_list).

Luckily, [dcode.fr has a decoder](https://www.dcode.fr/pgp-word-list), which gives us hexadecimal data that we can then decode using the "From Hex" recipe from [CyberChef](https://gchq.github.io/CyberChef), giving us our word.
