# SALLY Cover Challenge

This challenge can be found on the cover of issue 5. Spot the numbers in a vertical line in the bottom right.

## The Code

```
8 333 444 222 666 6 444 222 1 444 666 11111111 8 33 2 6 9 666 777 55
```

This is a fun one! [dcode's cipher identifier](https://www.dcode.fr/cipher-identifier) correctly finds it is SMS T9-style code.
If you're born before 2000, this might make sense, if not, tough luck, look it up youngling.

This decodes to a tficomic URL as usual, and the web page contains the flag.
