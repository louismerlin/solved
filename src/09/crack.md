# GLASSHEADS, pt. 2 Crack the \*\*\*\*\*\*

This challenge can be found on the inside of the cover of issue 9.

We see a list of five symbols, followed by a table of six by three symbols.

> Convert from base 5.
> 
> Use the row at the top for the order
>
> For example the circle is 0, plus is 1, square is 3, etc

These instructions seem simple enough. The first column, square waves circle, translates to 320. 320 in base 5 is 85. 85 is the charcode for U.

Repeating this logic for all six columns we get the word.
