# SALLY Cover Challenge

This challenge can be found on the cover of issue 4. Spot the dots right under the title.

## The Code

This looks an awful lot like [Braille](https://en.wikipedia.org/wiki/Braille) code, used by blind and visually impaired people.

Decoding it, we get a URL that leads us to a bonus flag.

Note that there is an extra character between the address and the "/" that needs to be removed.
